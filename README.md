# ShopEase — One-Page E-Commerce Website

A fully functional single-page e-commerce website built with vanilla HTML, CSS, and JavaScript.

## Features

- **Product Catalog** — 8 products across multiple categories with emoji thumbnails
- **Cart Drawer** — slide-in cart with quantity controls and running total
- **Login / Sign Up** — tabbed modal with client-side validation
- **3-Step Checkout** — Shipping → Payment → Order Review
- **Order Confirmation** — success modal with generated order ID
- **Toast Notifications** — feedback on cart and auth actions
- **Responsive Design** — works on mobile, tablet, and desktop

## Getting Started

No build step required. Just open `index.html` in a browser.

```bash
open index.html
```

## File Structure

```
├── index.html   # Page structure and modals
├── style.css    # All styles (CSS variables, animations, responsive)
└── app.js       # Product data, cart, auth, checkout logic
```
